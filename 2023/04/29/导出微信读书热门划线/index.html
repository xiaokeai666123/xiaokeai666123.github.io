一键导出微信读书热门划线

Posted on 04/29/2023 Edited on 08/07/2023 In tech

微信读书虽然书目有限，但它的优点也是显而易见的：

  • 多端同步
  • iOS 端最优秀的中文 tts（text-to-speech 文字转语音）

基于这两点，我经常用它来读一些“不需要动脑子”的书。有些时候，甚至是直接把书传进来，然后用它自带的 tts 听书。

但有一个痛点是，这些“不需要动脑子”的书很多是在我做运动或者带娃的时候读（听）的，所以很难在同一时间做笔记，甚至连划线都做不到。

后来我想到一个办法：用其他读者的“热门划线”来作为这本书的 summary。这当然是一个折中的办法，hit 到其他人的 points，并不一定会和我产生共鸣；对我有所触动的 points 可能也并不在热门划线之列。但是这些书是“不需要动脑子”的，所以并不需要精读和深入思考，何况经过大多数人的 vote，这些片段应该也足以概括它的精华。

于是，在周五的晚上，趁娃睡着之后，我琢磨了一下如何自动化实现。

难点

  • 微信读书的热门划线只可以在移动端浏览，也就是说，在 web 端不能直接得到 response
  • 有些图书是付费图书，无法直接拿到这些书的 bookID，也就无法在 web 端 query 这些书的热门划线

如何拿到付费图书的 bookID

有一种办法可以拿到微信付费图书的 bookID。

首先找到这本付费图书，然后在移动端打开并查看它的热门划线；

然后点开任意一条热门划线，就能浏览它的上下文；

此时，微信读书会进行一个 5min 的倒计时，限制你查看这本付费图书的全部内容。但在这个时间段，你去 query 登录接口，就可以得到这本书的 bookID 了！
